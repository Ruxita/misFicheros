#Mis Ficheros es una aplicación desarrollada en Java con Swing, diseñada para la busqueda de archivos en el sistema operativo según su extensión.
El programa utiliza procesos del sistema ( ProcessBuilder) para realizar búsquedas realies en las unidades del equipo, mostrando los resultados en una 
interfaz gráfica sencilla y funcional. 

#El proyecto se ha construido utilizando las siguientes herramientas y tecnologías:
-Lenguaje de programacion: Java
-Entono de desarrollo (IDE) : Eclipse 2025-12 M1 (4.38.0 M1)
-Libreria gráfica: Swing (Gui)

#Instalación y ejecución Requisitos previos

Eclipse for Java Developers
JDK 17 (o compatible con tu versión de Eclipse)
Plugin WindowBuilder instalado en Eclipse

#Pasos para el Setup

Clonar el repositorio. Abre tu terminal o Git Bash y ejecuta el siguiente comando para clonar el proyecto:
git clone https://github.com/Ruxita/misFicheros.git

Abrir el proyecto en Eclipse: 
ve a File-> Import -> Existing Projects into Workspace. 
selecciona la carpeta del repositorio clonado. 

Ejecuta la clase principal.
Abre el archivo GUIBuscador.java. 
Haz clic en el botón Run (el triángulo verde) para compilar y ejecutar la aplicación.
